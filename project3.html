Mental Health Game

This mobile application is a mental health-focused game designed to educate and engage users. It includes learning content, quizzes to reinforce knowledge, and a reward system to motivate progress. Leaderboard features allow users to track achievements and compare with peers. The app aims to promote mental well-being through gamified learning, encouraging healthy habits and self-awareness in an engaging, interactive format.

IDE: Visual Studio Code

Programming Language: Java
